Subiecte

Studentii inregistrati la acest concurs trebuie sa rezolve toate subiectele.

1. Se dă un graf neorientat cu n (n<=100) vârfuri si m muchii prin lista muchiilor.

Afisați toate subgrafurile pe care le are. Fiecare subgraf va fi afișat astfel: - vârfurile lui, - matricea de adiacență.

2. Se dau două grafuri neorientate G1 și G2 cu n (n<=100) vârfuri și m1, respectiv m2 muchii prin listele muchiilor, în două fișiere graf1.in, respectiv graf2.in.

Verificați dacă unul dintre ele este graf parțial al celuilalt. Se va preciza dacă G1 este graf parțial al lui G2, dacă G2 este graf parțial al lui G1 sau NU dacă niciunul dintre ele este graf parțial al celuilalt.

3. Se dă un graf neorientat cu n (n<=100) vârfuri si m muchii prin lista muchiilor.

Afisați toate grafurile parțiale pe care le are. Fiecare graf parțial va fi afișat astfel: - numărul lui de ordine (al câtelea a fost generat), - lista munchiilor lui, - matricea de adiacență.

4. Realizați în C# un program care să permită crearea unei table de dimensiune n*n de tipul unei table de sah care să conțină n*n butoane.

Pe fiecare buton să se scrie linia și coloana pe care se află, iar la click pe el să fie scrise într-un MessageBox linia și coloana pe care se află.

5. Realizați în JavaScript un editor de fișiere Markdown.

Programul trebuie să permită alegerea următoarelor operații dintr-un meniu: - fisier nou, - deschidere, - salvare, - salvare ca, - iesire, - editare, - copiere, - taiere, - lipire, - selectare totala, - formatare, - font, - culoare, - aliniere.
